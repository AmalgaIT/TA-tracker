🔧 First-Time Setup

Connect the tracker to a shared Google Sheet for multi-user sync

1
Create a Google Sheet

Go to sheets.new, name it "TA Tracker", and share it with all TA specialists (Editor access).

2
Get your Sheet ID

Copy the long ID from the URL:
docs.google.com/spreadsheets/d/[SHEET ID]/edit

3
Create OAuth Client ID

In Google Cloud Console: enable Sheets API, create OAuth 2.0 credentials (Web app), add your page URL as an authorized origin. Copy the Client ID.
